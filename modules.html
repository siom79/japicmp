Project Modules

This project has declared the following modules:

Name Description
japicmp japicmp is a library that computes the differences between two versions of a jar file/artifact in order to ease the API documentation for clients/customers.
japicmp-testbase japicmp is a tool/maven plugin that compares two versions of a jar archive for binary compatibility.
japicmp-maven-plugin japicmp-maven-plugin is a maven plugin that computes the differences between two versions of a jar file/artifact in order to ease the API documentation for clients/customers.
japicmp-ant-task japicmp-ant-task is an Ant task that computes the differences between two versions of a jar file/artifact in order to ease the API documentation.
